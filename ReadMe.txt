Softwares necessários para rodar o sistema: Mysql, Xampp, (VsCode ou alguma outra IDE ou Editor de Código Fonte).

Para rodar o projeto da maneira correta é preciso que o esquema de diretórios estejam igual ao que foi desenvolvido. Então crie um arquivo 
chamado "TCC" no htdocs do xampp e clone este projeto para dentro da pasta "TCC" e extraia todos os arquivo de modo que só mostre as pastas "Model", "Controller", "View", "Img" e o arquivo "index.php". Depois de fazer este processo, você poderá excluir as pastas e os arquivos que não foram nomeados anteriormente.

No arquivo ConexaoBD.php dentro da camada "Model" configure as variáveis de ambiente de acordo com as configurações do seu banco de dados.
